# Домашние рецепты

PWA-приложение для хранения рецептов, планирования меню на неделю и формирования списка покупок.

## Что уже работает

- главная страница с блюдом дня, недельным меню, избранным и последними рецептами;
- карточки рецептов с фильтрами по категории, тегу, времени, избранному и ингредиенту;
- подробный рецепт в большом попапе;
- режим «Готовлю сейчас» с крупными шагами и таймерами;
- меню на неделю: завтрак, обед, ужин, перекус;
- добавление, замена и удаление рецептов в слотах меню;
- копирование прошлой недели и очистка текущей недели;
- автоматический список покупок на день или неделю;
- объединение одинаковых ингредиентов при совпадении единицы измерения;
- ручное добавление, редактирование, удаление и отметка «куплено»;
- копирование списка покупок текстом;
- PWA-манифест и service worker для установки на главный экран.
- семейные оценки для Артема, Тёмы, Ильи и Оксаны;
- бейджи происхождения рецепта: от мамы, от бабушки, семейный, из интернета, мой рецепт;
- история приготовлений с фото, датой, комментарием, участниками и заметкой на следующий раз;
- экран «Добавить рецепт» с ручным вводом, ссылкой и текстом перед подтверждением;
- раздел «Что есть дома» с базовыми продуктами и рекомендациями рецептов;
- кнопки «Не хочу», «Осталось на завтра», фильтры «детям нравится», «семейные хиты», «до 30 минут».

## Как открыть

Можно открыть файл `index.html` прямо в браузере.

Для полноценной PWA-установки на iPhone приложение нужно разместить на HTTPS-домене. После этого в Safari: «Поделиться» → «На экран Домой».

## Supabase

Файл `supabase-schema.sql` содержит структуру таблиц PostgreSQL под сущности из ТЗ:

- `recipes`
- `ingredients`
- `steps`
- `weekly_menu`
- `shopping_lists`
- `shopping_items`
- `people`
- `recipe_ratings`
- `cooking_history`
- `home_products`

Текущая версия работает в локальном режиме через браузерное хранилище, чтобы приложение можно было открыть сразу без ключей. Для продакшена следующий шаг: подключить Supabase URL и anon key, затем заменить локальное сохранение в `app.js` на запросы к таблицам Supabase.

## Быстрая синхронизация Supabase

В `app.js` подключены:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`

Для работы облачного хранения нужно выполнить `supabase-schema.sql` в Supabase SQL Editor. Приложение сохраняет весь текущий state в таблицу `app_state`, поэтому рецепты, меню, покупки, оценки, история и продукты дома будут синхронизироваться между устройствами. Если Supabase недоступен, приложение продолжит работать локально.
